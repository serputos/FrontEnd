▶️ Introducción

Todos los navegadores incorporan funcionalidades que permiten a los desarrolladores probar sus implementaciones. En esta asignatura nos centraremos en los dos que a nuestro parecer ofrecen la mejor experiencia durante el desarrollo y depuración de páginas web. Son Chrome de Google y Firefox de Mozilla.

Ambos exploradores ofrecen una versión exclusiva para desarrolladores: Chrome Cannary y Firefox Developer Edition.

Es interesante de igual forma investigar y probar las herramientas de otros navegadores como Safari o Edge.

🚀 Chrome

Chrome DevTools es un conjunto de herramientas de creación web y depuración integrado en Google Chrome. Se pueden usar para iterar y depurar un sitio web, y para mejorar la implementación.

Las DevTools se pueden abrir de 3 formas distintas:

  • En el Menú de Chrome, seleccionar More Tools > Developer Tools.
  • Click con el botón secundario en un elemento de la página y seleccionar Inspect.
  • Usar las combinaciones de teclas Ctrl+Mayúscula+I (Windows) o Cmd+Opción+I (Mac).

🦊 Firefox

Las herramientas de Firefox también te permiten examinar, editar y depura HTML, CSS y JavaScript.

Para abrir las herramientas de desarrollo de Firefox se puede:

  • Elegir la opción de "Inspector" en el menú "Web Developer" (que es un submenú en el menú "Herramientas" en Mac)
  • Pulsar Ctrl-Shift-C (Cmd + Opción + C en Mac OS X y Linux)
  • Hacer click derecho en un elemento de una página web y seleccionar "Inspeccionar elemento"
